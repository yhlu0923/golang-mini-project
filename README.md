# golang-mini-project

## Our ideas

### Topics
-	web service
-	data processing

### Descriptions

建立一個line 聊天機器人
使用者只要輸入今天想看的東西(字串)
我們就會去找到相關的圖片傳送給使用者
	
	e.g 
假設使用者想看「狗」，只要輸入，「抽狗」
我們的聊天機器人就會找一張狗的圖片傳送給使用者
想看科基就「抽科基」...etc
